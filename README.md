TecnoRepuestos S.A. — Sistema de Gestión de Inventario y Ventas

Este repositorio contiene la base del sistema desarrollado para TecnoRepuestos S.A., una empresa dedicada a la venta de repuestos automotrices. Aquí se organiza la estructura inicial del proyecto siguiendo buenas prácticas de arquitectura.

Tecnologías

Node.js + Express

Base de datos relacional (MySQL o PostgreSQL)

Arquitectura en capas (rutas, controladores, servicios, acceso a datos)


Cómo ejecutar

npm install
npm start
